基于github page+hexo搭建个人域名博客网址

域名购买

选在一个域名厂商购买域名，我本人是在百度云购买的www.zcx.com 价格还不贵一年几十块

注册github账号

安装hexo git

  • 在bolg文件夹按住shift键点击鼠标右键，点击在此处打开命令行。执行命令：npm install hexo-cli -g #安装hexo
  • 在命令行中 输入命令：npm install hexo-deployer-git –save #部署插件
  • 下载git 客户端，按照默认安装安装完成后，在之前的blog文件夹中点击鼠标右键点击git bash
  • 在git终端中输入hexo init xxx #初始化网站
  • 进入blog文件夹中，在根目录中打开 _config.yml 文件，最后一行添加：

hexo g
hexo d
`
自此github page 博客搭建完成。

域名解析

  • 进入你购买域名厂商的后台，找到域名解析。添加解析规则
    @ cname xxxx.github.io
    www cname xxxx.github,io

  • 在blog文件夹中的source文件夹中添加文件：CNAME(全部是大写),在文件夹中输入：购买的域名

至此，博客搭建完成，输入域名就可以进入我的个人博客了
